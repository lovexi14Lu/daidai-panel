# v2.0.5 更新日志

## 新增功能

### Magisk 面具模块支持
- 新增 Android Magisk 面具模块，可以在 Root 的安卓手机上安装呆呆面板
- 支持 ARM64 和 ARM 32位 两种架构
- 开机自动启动服务
- 数据存储在 `/data/adb/daidai-panel` 目录
- 默认端口 5700

## 安装方式

1. 下载 Magisk 模块 zip 文件
2. 在 Magisk Manager 中安装
3. 重启系统
4. 访问 `http://localhost:5700`

## 文件说明

- `daidai-panel-magisk-{version}-arm64.zip` - ARM64 架构（推荐）
- `daidai-panel-magisk-{version}-arm.zip` - ARM 32位架构
